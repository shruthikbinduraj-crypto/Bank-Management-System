# Bank Management System

A console-based Bank Management System developed in C using dynamic memory allocation and file handling.

## Features

- Add Account
- Display Accounts
- Search Account
- Deposit Money
- Withdraw Money
- Delete Account
- Save Accounts to File
- Load Accounts from File

## Concepts Used

- Structures
- Dynamic Memory Allocation
- File Handling
- Functions
- Arrays
- Pointers
